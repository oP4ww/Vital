# Vital

Vital is a client-side Fabric mod for Minecraft Java Edition `26.1.1`.

It renders clean target-health hearts above living entities and shows floating combat popups without requiring the server to install anything.

## Features

- Heart-based health indicator above living entities.
- Optional exact HP values.
- Floating damage and healing popups.
- Recent-hit trigger mode.
- Look-target trigger mode.
- Editable keybinds in the `Vital` category.
- JSON config support.
- Optional Mod Menu config screen.
- Client-side only.

## Why use Vital?

Vital is aimed at players who want readable combat feedback in survival without a noisy HUD or chat output.

## Requirements

- Minecraft `26.1.1`
- Fabric Loader
- Fabric API
- Java `25`
- Mod Menu is optional

## Build

```bash
gradle clean build
```

Built jar:

```text
build/libs/vital-1.0.0+26.1.1.jar
```
