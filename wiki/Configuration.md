# Configuration

Vital supports an in-game config screen and JSON config storage.

## Config Access

You can open settings through:

- the Vital keybind for opening settings
- Mod Menu, if installed

## Main Settings

### General

- Mod enabled
- Health indicators enabled
- Damage popups enabled
- Healing popups enabled

### Trigger Modes

- Show after recent hit
- Show when looking at target
- Recent-hit display time

### Visual Settings

- Heart row enabled
- Exact values enabled
- Health indicator opacity
- Popup opacity
- Render distance
- Maximum tracked entities

## JSON Config

Config file path:

```text
config\vitalpopups.json
```

## Recommended Setup

For a cleaner PvP/PvE experience:

- enable heart row
- enable look-target mode
- enable recent-hit mode
- set a medium opacity for the indicator
- use popups sparingly if you prefer less visual clutter
