# Controls

Vital adds editable keybinds inside Minecraft's normal keybind menu.

## Where to Change Keys

Open:

**Options -> Controls -> Key Binds**

Search for:

**Vital**

## Available Actions

- Toggle Vital
- Toggle target hearts
- Toggle damage popups
- Toggle healing popups
- Toggle hit-trigger mode
- Toggle look-target mode
- Toggle heart row
- Toggle HP numbers
- Open Vital settings

## Notes

All keybinds are intended to be editable like other mod keybinds.
If a key conflicts with another mod, simply rebind it in the keybind menu.
