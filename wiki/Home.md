# Vital Wiki

Welcome to the official wiki for **Vital**.

Vital is a client-side Fabric mod for Minecraft Java Edition that shows target health in a clean, readable way with floating combat feedback.

## Features

- Heart-based target health indicator
- Floating combat popups
- Hit-trigger display mode
- Look-at-target display mode
- Editable keybinds
- In-game config screen
- Optional Mod Menu integration
- Lightweight client-side design
