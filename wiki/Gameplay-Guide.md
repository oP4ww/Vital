# Gameplay Guide

Vital is designed to be readable in normal survival gameplay without turning the screen into a full HUD replacement.

## Basic Use

Look at a mob or player to reveal health when look-target mode is enabled.

Hit an entity to reveal health when recent-hit mode is enabled.

Floating popups show combat feedback during fights.

## Best Use Cases

- survival combat
- PvE mob tracking
- boss encounters
- casual target checking

## Cleaner Experience Tips

- keep only heart display enabled
- disable exact values if you want less noise
- raise popup opacity for easier reading
- use look-target mode for a minimal UI feel
