# Description Ideas

## Safer, cleaner tone

Vital adds readable target-health hearts and floating combat feedback to Minecraft without turning the screen into a large permanent HUD.

## More gameplay-focused tone

Track target health quickly during survival combat with clean in-world hearts, hit-trigger display modes, and floating damage feedback.

## More technical tone

A client-side Fabric overlay that renders target health and combat popups in-world, with JSON config, editable keybinds, and optional Mod Menu support.
