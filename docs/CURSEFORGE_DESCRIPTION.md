# Vital

Vital is a client-side Fabric mod for Minecraft Java Edition `26.1.1` that adds clean target-health hearts and floating combat popups.

## Features

- Heart-based target health display.
- Optional exact health values.
- Floating damage popups.
- Floating healing popups.
- Recent-hit trigger mode.
- Look-target trigger mode.
- Editable keybinds in a dedicated `Vital` controls category.
- JSON config support.
- Optional Mod Menu integration.
- No server-side install required.

## Best use case

Vital is intended for survival gameplay where you want fast visual feedback on target health without chat text or a large permanent HUD.

## Requirements

- Fabric Loader
- Fabric API
- Minecraft `26.1.1`
- Java `25`

## Optional

- Mod Menu for an in-game config screen

## Config path

`config/vitalpopups.json`

## Keybinds

Edit them in:

`Options -> Controls -> Key Binds -> Vital`
