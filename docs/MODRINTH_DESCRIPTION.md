# Vital

Vital is a clean client-side Fabric mod for Minecraft Java Edition `26.1.1` that shows target health in-world and gives immediate combat feedback without chat spam.

## What it does

- Shows a heart-based health indicator above living entities.
- Can show optional exact HP values.
- Shows floating damage popups.
- Shows floating healing popups.
- Can reveal health only after the target is hit.
- Can reveal health only while you are looking at the target.
- Includes editable keybinds grouped under the `Vital` controls category.
- Saves settings to a JSON config file.
- Supports Mod Menu for an in-game config screen.

## Why use it

Vital is built for players who want readable combat information during normal gameplay while keeping the screen cleaner than large permanent HUD overlays.

## Client-side only

This mod is client-side. The server does not need to install it.

## Compatibility

- Minecraft `26.1.1`
- Fabric
- Java `25`

## Controls

You can edit all Vital keybinds in:

`Options -> Controls -> Key Binds -> Vital`

## Notes

- Fabric API is required.
- Mod Menu is optional.
- Config file path: `config/vitalpopups.json`
